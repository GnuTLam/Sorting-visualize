## Giới thiệu
Sản phẩm được xây dựng nhằm trực quan hóa các thuật toán sắp xếp thông dụng. Các thuật toán được mô tả dưới dạng hoạt ảnh để người dùng có thể hình dung một cách dễ dàng.

### Mô tả 
##### Các thuật toán sắp xếp
- Sắp xếp lựa chọn
- Sắp xếp nổi bọt
- Sắp xếp chèn
- Sắp xếp nhanh
- Sắp xếp trộn
